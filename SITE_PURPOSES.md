# Documentation Sites - Purposes and Scope

This document describes all 13 documentation sites in the Zudoku monorepo, including their purpose, target audience, and content scope.

## 1. brightforest.io

**Purpose**: Primary platform documentation for BrightForest's ecosystem  
**Audience**: Developers, product managers, technical decision-makers  
**Content Scope**:
- Platform overview and architecture
- API documentation
- Integration guides
- Best practices and tutorials
- Enterprise features
- Security and compliance docs

## 2. brightforestx.com

**Purpose**: Extended/enterprise version documentation  
**Audience**: Enterprise customers, solution architects, DevOps teams  
**Content Scope**:
- Enterprise-specific features
- Advanced configuration
- Multi-tenant setup
- Custom deployment options
- SLA and support documentation
- Migration guides from standard to enterprise

## 3. brightpath.ai

**Purpose**: AI-powered development pathfinding and guidance platform  
**Audience**: Developers learning AI, technical leads, AI practitioners  
**Content Scope**:
- AI/ML development guides
- Model integration tutorials
- Best practices for AI implementation
- Use case examples
- API reference for AI services
- Performance optimization guides

## 4. pathx.ai

**Purpose**: Specialized AI pathfinding tools and frameworks  
**Audience**: AI engineers, data scientists, ML researchers  
**Content Scope**:

- Framework documentation

- Benchmark results
- Contribution guidelines

## Common Themes Across All Sites

All sites share these requirements:
- Responsive design (mobile, tablet, desktop)
- Dark mode support
- Search functionality
- SEO optimization
- Accessibility (WCAG 2.1 AA)
- Performance (Lighthouse 90+)
- Analytics integration
- Error handling and 404 pages
- Privacy policy and legal pages
