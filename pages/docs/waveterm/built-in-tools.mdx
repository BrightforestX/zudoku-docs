# Built-In Tools in GitHub Copilot Chat for VS Code

This document provides an overview of the built-in tools available in GitHub Copilot Chat for Visual
Studio Code. These tools are accessible via the "Configure Tools" window and enable a wide range of
coding, workspace, and automation tasks. Descriptions are based on the interface screenshots and may
be truncated where the original UI was cut off.

## Tool List

### agent

Delegate tasks to other agents.

### awaitTerminal

Wait for a background terminal command to complete. Returns the output.

### changes

Get diffs of changed files.

### codebase

Find relevant file chunks, symbols, and other information via semantic search.

### createAndRunTask

Create and run a task in the workspace.

### createDirectory

Create new directories in your workspace.

### createFile

Create new files.

### createJupyterNotebook

Create a new Jupyter Notebook.

### edit

Edit files in your workspace.

### editFiles

Edit files.

### editNotebook

Edit a notebook file in the workspace.

### execute

Execute code and applications on your machine.

### fetch

Fetch the main content from a web page. You should include the URL of the page.

### fileSearch

Find files by name using a glob pattern.

### getNotebookSummary

Returns the list of the Notebook cells along with their details.

### getTerminalOutput

Get the output of a terminal command previously started with runInTerminal.

### githubRepo

Search a GitHub repository for relevant source code snippets. You can specify the repo.

### killTerminal

Kill a terminal by its ID. Use this to clean up terminals that are no longer needed.

### listDirectory

List the contents of a directory.

### problems

Check errors for a particular file.

### read

Read files in your workspace.

### readFile

Read the contents of a file.

### readNotebookCellOutput

Read the output of a previously executed cell.

### runInTerminal

Run commands in the terminal.

### runNotebookCell

Trigger the execution of a cell in a notebook file.

### runSubagent

Run a task within an isolated subagent context to enable efficient organization.

### runTests

Run unit tests (optionally with coverage).

### search

Search files in your workspace.

### searchResults

Get the results of the search view.

### terminalLastCommand

Get the last command run in the active terminal.

### terminalSelection

Get the current selection in the active terminal.

### testFailure

Include information about the last unit test failure.

### textSearch

Search for text in files by regular expression.

### todo

Manage and track todo items for task planning.

### usages

Find references, definitions, and other usages of a symbol.

### vscode

Use VS Code features.

### web

Fetch information from the web.
