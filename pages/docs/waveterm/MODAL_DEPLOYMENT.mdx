# Modal Deployment Guide

## Overview

The CrewAI Orchestrator is deployed on Modal.com for serverless, scalable execution.

- Status: Active

## Endpoints

### Health Check

```bash
GET /health
```

Returns service health status

### Status

```bash
GET /api/v1/status
```

Returns API version and enabled features

### Analyze Request

```bash
POST /api/v1/analyze
Content-Type: application/json

{
  "input": "Show me my calendar for next week",
  "provider": "google_calendar",
  "user_id": "user_123"
}
```

### Execute Action

```bash
POST /api/v1/execute
Content-Type: application/json
{
  "action_id": "list_events",
  "parameters": {
    "start_date": "2026-02-18",
    "end_date": "2026-02-25"
  }
}
```

### List Providers

```bash
GET /api/v1/providers
```

## Deployment Commands

### Deploy to Staging

```bash
modal deploy src/crewai_orchestrator/modal_app.py
```

### Test Endpoints

```bash
# Health check
curl https://brightforestx--crewai-orchestrator-staging-fastapi-app.modal.run/health

# Status
curl https://brightforestx--crewai-orchestrator-staging-fastapi-app.modal.run/api/v1/status

# Analyze
curl -X POST https://brightforestx--crewai-orchestrator-staging-fastapi-app.modal.run/api/v1/analyze \
  -H "Content-Type: application/json" \
  -d '{"input": "Show me my tasks", "provider": "asana"}'

# Providers
curl https://brightforestx--crewai-orchestrator-staging-fastapi-app.modal.run/api/v1/providers
```

## Environment Variables

- `ANTHROPIC_API_KEY`: Anthropic API key
- `NANGO_SECRET_KEY`: Nango integration key
