## Render All to PNG

```bash
cd docs/flows
for f in *.d2; do
  d2 "${f}" "${f%.d2}.png"
done
```

Or from project root:

```bash
cd /Users/clifforddalsoniii/Documents/Brightforest/projects/controller/brightpath_swarm/docs/flows
for f in *.d2; do d2 "${f}" "${f%.d2}.png"; done
```

## D2 Version

Requires D2 0.7.0+ with `sequence_diagram` support. Install: `brew install d2`.
