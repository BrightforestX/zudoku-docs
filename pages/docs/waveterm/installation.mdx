# Installation

Waveterm is available for Mac, Windows, and Linux.

## Mac Installation

### Download

Download the latest version from
[github.com/wavetermdev/waveterm/releases](https://github.com/wavetermdev/waveterm/releases).

### Install

1. Open the downloaded `.dmg` file
2. Drag Waveterm into your Applications folder

### Verify Installation

Open Waveterm from your Applications folder to verify it installed correctly.

## Windows Installation

### Download

Download the latest version from
[github.com/wavetermdev/waveterm/releases](https://github.com/wavetermdev/waveterm/releases).

### Install

Run the `.exe` installer and follow the prompts to install Waveterm.

### Verify Installation

Search for Waveterm in the Start Menu to verify it installed correctly.

## Linux Installation

### Snap

Install Waveterm using Snap:

```bash
sudo snap install waveterm --edge
```

### AppImage

1. Download the latest AppImage from
   [github.com/wavetermdev/waveterm/releases](https://github.com/wavetermdev/waveterm/releases)
2. Make the AppImage executable: `chmod +x Waveterm-*.AppImage`
3. Run the AppImage: `./Waveterm-*.AppImage`

### Verify Installation

Run Waveterm from your applications menu or command line to verify it installed correctly.
