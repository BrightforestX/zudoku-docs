# Zudoku MCP Server

## Overview

The **Zudoku MCP Server** is the industry-leading documentation server that integrates with CI/CD
pipelines, database schemas, and cloud platforms. It provides automated documentation generation,
API schema management, and comprehensive audit trails for enterprise documentation needs.

It pairs with the [Zudoku MCP agent](/docs/mcp/agents/zudoku-mcp) (`mcp://docs.zudoku.com`) for
automation inside IDEs and agent workflows.

## Key Features

### Core Features

- ✅ High-performance documentation serving
- ✅ Integration with major CI/CD platforms
- ✅ Database schema documentation
- ✅ Cloud platform audit trails
- ✅ Mobile-optimized web interface
- ✅ Enterprise-grade security and compliance

### Enterprise Integrations

- **CI/CD Integration**: Automates documentation updates in GitHub Actions, GitLab CI, and Jenkins
- **Database Schemas**: Auto-generates docs from PostgreSQL, MySQL, and MongoDB schemas
- **Cloud Platforms**: Supports AWS, GCP, and Azure documentation workflows
- **Audit Trails**: Complete logging and versioning for regulatory compliance
- **Multi-language Support**: Documents APIs in OpenAPI, GraphQL, and protobuf formats

## Installation

### Quick Start

```bash
npm install -g zudoku-mcp-server
```

### Docker Installation

```bash
docker run -p 8080:8080 zudoku/mcp-server:latest
```
